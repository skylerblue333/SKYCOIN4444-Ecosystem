### Mobile Experience

- **Desktop:** Full dropdown navigation (≥768px)
- **Tablet:** Simplified navigation with icons (≥480px)
- **Mobile:** Bottom nav + hamburger menu (<480px)

## Page Accessibility

All 699 pages are accessible via:
1. **Direct URL:** `/pagename` (lowercase)
2. **Top Navigation:** Category dropdown → Page link
3. **Bottom Navigation:** Quick access or "More" menu
4. **Search:** Global search functionality
5. **Breadcrumbs:** Navigation trail on each page

## Performance Optimization

- **Lazy Loading:** All pages use React lazy() for code splitting
- **Route-based Splitting:** Each page loads independently
- **Caching:** Browser caching for static assets
- **CDN:** Content delivery network for media
- **Compression:** Gzip compression for all assets
